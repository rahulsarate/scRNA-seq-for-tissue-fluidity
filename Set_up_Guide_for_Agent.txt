12.1 QC Signoff Checklist (~30 items)
 FastQC per-base quality >Q30
 Adapter content <5%
 GC content normal distribution
 Duplication levels acceptable
 Read count per sample >10M (bulk) / >1000 cells (single-cell)
 Mapping rate >70% (known genome)
 rRNA contamination <10%
 Library complexity adequate
etc.
12.2 DE Analysis Signoff (~25 items)
 PCA shows expected sample grouping
 No outlier samples (cook's distance)
 Appropriate model design (including batch if needed)
 Independent filtering applied
 Log2FC shrinkage applied
 Multiple testing correction (BH/FDR)
 Significant genes at FDR <0.05 reported
 Results saved as CSV + RDS
etc.

Section 14: Tips & Lessons Learned
10 practical tips:

Always check strandedness before counting
Use salmon for speed, STAR for junction discovery
DESeq2 for small samples (<20), edgeR for large
Always save sessionInfo() for reproducibility
Use RDS format not CSV for R objects (preserves types)
Check batch effects BEFORE DE analysis
Use lfcShrink (apeglm) not raw log2FC
PCA is your best friend — look at it first
Don't filter genes before DESeq2 (it does its own filtering)
Version control your scripts, not your data
